## Instalación

1. Clona el repositorio:
    ```bash
    git clone https://github.com/usuario/proyecto.git
    cd proyecto
    ```
2. Instala las dependencias:
    ```bash
    npm install
    # o
    pip install -r requirements.txt
    ```

Ejecuta la aplicación con:
```bash

```
